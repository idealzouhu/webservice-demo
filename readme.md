# 项目说明

本项目为客户投诉管理，包括客户投诉填写服务、投诉处理服务、通知用户服务。本项目实现三个服务以及相应的维护界面，其中投诉处理服务需要读取客户投诉填写服务的数据。

其中，customer-complaint 为客户端，customer-server为服务端



# 环境配置

`customer-complaint`只能使用 Oracle jdk 8 

`customer-server`可以使用jdk8以上的任意版本
